## 🚀 QUICK START (3 Steps)

### Step 1: Replace Backend
```bash
# Copy new server
cp server-complete.js backend/server.js
```

### Step 2: Update Frontend
```bash
# Copy updated files
cp App-complete.jsx frontend/src/App.jsx
cp Dashboard-complete.jsx frontend/src/Dashboard.jsx
cp PatientRegistration-complete.jsx frontend/src/PatientRegistration.jsx

# Add new components
cp AddDoctor.jsx frontend/src/
cp PatientsList.jsx frontend/src/
cp BookAppointment.jsx frontend/src/
cp AppointmentHistory.jsx frontend/src/
```

### Step 3: Run
```bash
# Terminal 1 - Backend
cd backend && npm run dev

# Terminal 2 - Frontend
cd frontend && npm start
```

**Done! Access at http://localhost:3000** ✅

## 🔧 API ENDPOINTS (All Working)

### Patients
- `GET /api/patients` - Get all patients
- `POST /api/patients` - Register patient
- `GET /api/patients/:id` - Get single patient
- `PUT /api/patients/:id` - Update patient

### Doctors
- `GET /api/doctors` - Get all doctors
- `POST /api/doctors` - Add doctor
- `GET /api/doctors/:id` - Get single doctor
- `PUT /api/doctors/:id` - Update doctor

### Appointments
- `GET /api/appointments` - Get all appointments
- `POST /api/appointments` - Book appointment
- `GET /api/appointments/patient/:id` - Get patient appointments
- `PUT /api/appointments/:id` - Update appointment
- `DELETE /api/appointments/:id` - Cancel appointment

### Dashboard
- `GET /api/dashboard/statistics` - Get all stats
- `GET /api/health` - Check API status

## 📞 TROUBLESHOOTING QUICK HELP

| Issue | Solution |
|-------|----------|
| "Cannot connect to server" | Make sure backend is running: `npm run dev` |
| "No patients in dropdown" | Register a patient first from UI |
| "Forms not submitting" | Check all * required fields are filled |
| "404 on API call" | Verify backend server URL is correct |
| "Appointment booking fails" | Need at least 1 patient AND 1 doctor first |
| "Port already in use" | Change PORT: `PORT=5001 npm run dev` |
